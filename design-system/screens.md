# Screens

## Child home

Greeting, avatar, level/XP/streak metrics, daily goal, one Continue CTA, current
route preview, nearest reward, and stable bottom navigation.

## Learning path

Vertical, lightly winding path grouped by units. Current node is visually
dominant; future nodes remain visible so the next milestone feels reachable.

## Task

Close control, lesson progress, concise question, large answer options, one
Check CTA, then inline child-safe feedback. Portrait and landscape are both
supported. Detailed help is progressively disclosed.

## Lesson result

Correct count, XP, elapsed time, streak, earned reward, and one Continue CTA.

## Goals, rewards, profile

Goals show only today's actions. Rewards distinguish earned and upcoming.
Profile combines identity, level progress, learned topics, and parent pairing.
