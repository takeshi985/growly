# Typography

Growly uses the platform sans-serif during MVP to avoid font loading and
licensing risk. A branded rounded family can replace it later through tokens.

| Role | Size | Weight | Use |
|---|---:|---:|---|
| Display | 32 | 800 | Celebration and major greeting |
| Headline | 26 | 800 | Screen title and task question |
| Title | 20 | 750 | Cards and sections |
| Body | 16 | 500 | Instructions and explanations |
| Label | 13 | 700 | Metrics and navigation |

Body text never uses a light weight. Text wraps before truncation unless a
compact metric explicitly requires one line. System text scaling is preserved.
