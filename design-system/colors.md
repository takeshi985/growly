# Colors

Primitive values are exposed to Flutter only through semantic roles.

| Role | Value | Purpose |
|---|---:|---|
| brand | `#167C80` | Primary actions and current progress |
| brand-pressed | `#0E5E62` | Button depth and pressed state |
| brand-soft | `#DDF4F1` | Selected and educational surfaces |
| accent | `#F49B45` | Streak and warm emphasis |
| reward | `#F4C84A` | XP, stars, earned rewards |
| imagination | `#8267C7` | Logic and creative content |
| success | `#2F8F65` | Correct answers with icon/text |
| help | `#3979B8` | Hints and explanations |
| danger | `#C9505A` | Adult-only destructive actions |
| canvas | `#F7F4EC` | App background |
| surface | `#FFFFFF` | Cards and sheets |
| ink | `#193334` | Primary text |
| ink-muted | `#607273` | Supporting text |
| outline | `#D9E1DD` | Borders and separators |

Do not use reward yellow for body text. Functional states always include an
icon and label. Duolingo's palette and exact green are intentionally avoided.
