# Navigation

Child mode has four stable top-level destinations:

1. Учиться — home and learning path.
2. Цели — today's small achievable actions.
3. Награды — earned stars and badges.
4. Профиль — progress summary and parent connection.

The bottom bar is persistent only at top level. Tasks and pairing are pushed as
focused routes with an explicit close/back control. System back must always
return to the previous visible state and never reveal a blank screen.

Parent mode remains visually calmer and uses ordinary pushed routes until its
information architecture grows enough to justify a separate navigation shell.
