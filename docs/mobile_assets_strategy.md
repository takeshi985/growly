# Стратегия мобильных ресурсов

Сейчас прототип использует emoji как временные понятные маркеры. Следующий визуальный слой заменит их оригинальными иллюстрациями Growly.

- `assets/tasks/math/apple.png`
- `assets/tasks/math/basket.png`
- `assets/characters/growly_sprout.png`

Первая партия: 10–20 иллюстраций к основным заданиям. В репозиторий не добавляются тяжёлые исходники; перед публикацией изображения оптимизируются. Визуальные детали должны поддерживать задание, а не отвлекать ребёнка.
